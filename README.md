# koa2-plus

`koa2` 开发web项目**API**接口的模板仓库

## 运行

1. 安装依赖
    ```
    $ pnpm install
    ```

2. 启动服务
    ```
    $ pnpm start
    ```

3. 代码格式化
    ```
    $ pnpm lint
    ```
